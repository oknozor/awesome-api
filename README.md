# tiny-api
minimalist api build  with express js to demonstrate the power of systemd

## Requirement :

To run tiny-api you need nodeJS, npm and Mongodb. 

## Installation : 

1. clone the repo : 
```
$ git@github.com:oknozor/tiny-api.git
```
2. init npm : 
```
$ cd tiny-api
$ npm init
```
3. install the dependencies : 
```
$ npm install --save
```

4. run : 
```
node server.js
```
